### Anotaciones de Spring Utilizadas

Las siguientes anotaciones de Spring fueron detectadas en el código base:

- **@RestController** - Marca las clases como controladores REST
- **@RequestMapping** - Mapea las solicitudes HTTP a métodos manejadores
- **@GetMapping** - Maneja las solicitudes HTTP GET
- **@PostMapping** - Maneja las solicitudes HTTP POST
- **@PutMapping** - Maneja las solicitudes HTTP PUT
- **@DeleteMapping** - Maneja las solicitudes HTTP DELETE
- **@PatchMapping** - Maneja las solicitudes HTTP PATCH
