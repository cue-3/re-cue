## [0.0.9] - 2025-11-29

### Added
- Initial release to PyPI
- **Comprehensive reverse engineering toolkit** for analyzing codebases
- **Multi-framework support**:
  - Java: Spring Boot (full support)
  - Ruby: Rails (full support)
  - Python: Django, Flask, FastAPI (in development)
  - Node.js: Express, NestJS (in development)
  - .NET: ASP.NET Core (planned)
- **Documentation generators**:
  - Feature specifications (`spec.md`)
  - Implementation plans (`plan.md`)
  - Data model documentation (`data-model.md`)
  - API contracts (OpenAPI 3.0 `api-spec.json`)
  - Use case analysis (`use-cases.md`)
  - Business process diagrams (`diagrams.md`)
- **Advanced analysis capabilities**:
  - Actor identification and classification
  - System boundary detection
  - API endpoint discovery
  - Data model extraction
  - Business process visualization
- **Caching system** for improved performance
- **CLI interface** with comprehensive options
- **Template system** using Jinja2 for customizable output

### Performance
- Intelligent caching reduces analysis time on subsequent runs
- Multiprocess support for parallel analysis of large codebases

### Documentation
- Comprehensive user guides
- Developer documentation
- Framework-specific guides
- API reference documentation
