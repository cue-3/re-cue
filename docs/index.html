Quick Start

  1. Clone the Repository

    git clone https://github.com/quickcue3/specify-reverse.git cd specify-reverse

  2. Install into Your Specify Project

    ./install.sh /path/to/your/specify/project

    This integrates the toolkit into your existing Specify structure.

  3. Reverse Engineer Your Codebase

    # Generate complete documentation ./.specify/scripts/bash/reverse-engineer.sh --spec --plan --data-model --api-contract # Or use GitHub Copilot /speckit.reverse

GitHub Copilot Integration

GitHub Copilot will automatically:

  • Run specification generation
  • Create implementation plans
  • Analyze data models
  • Generate API contracts
  • Provide status updates for each step
