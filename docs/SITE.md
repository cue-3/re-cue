# RE-cue GitHub Pages Site

This directory contains the GitHub Pages site for RE-cue.

## Local Development

To test the site locally:

```bash
# Open index.html in your browser
open index.html
```

## Structure

- `index.html` - Main landing page
- `assets/css/style.css` - Stylesheet with dark blue, light blue, and white theme

## Colors

- Dark Blue: `#0a1929`
- Medium Blue: `#1e3a5f`
- Light Blue: `#4a90e2`
- Accent Blue: `#64b5f6`
- White: `#ffffff`

## GitHub Pages Setup

The site will be automatically published to: `https://quickcue3.github.io/re-cue/`

To enable GitHub Pages:
1. Go to repository Settings
2. Navigate to Pages section
3. Set Source to main branch
4. Select root directory
5. Save

The site should be live within a few minutes.
