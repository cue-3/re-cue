Installation

Quick Install

cd reverse-engineer-python
./install-python.sh

Run Analysis

# Generate all phase documents
python3 -m reverse_engineer --use-cases --path ~/projects/my-app

# Generate specific documents
python3 -m reverse_engineer --spec --plan --data-model

Documentation Outputs

Phase 1: Structure

API endpoints, data models, UI views, services, and features

Phase 2: Actors

User roles, permissions, and external system interactions

Phase 3: Boundaries

System boundaries, subsystems, layers, and architecture

Phase 4: Use Cases

Business processes, workflows, and use case scenarios
