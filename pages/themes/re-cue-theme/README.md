# RE-cue Hugo Theme

This is a custom Hugo theme for the RE-cue reverse engineering toolkit website.

## Features

- Responsive design
- Dark blue, light blue, and white color scheme
- Smooth animations and transitions
- Mobile-friendly navigation
- Section-based layout (Hero, Features, How It Works, Installation, Documentation)

## Usage

This theme is automatically used by the Hugo site configuration in `hugo.toml`.
